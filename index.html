Вопросы и ответы

Часто задаваемые вопросы

Не нашли ответ? Напишите нам, и мы поможем!

Нужны ли технические навыки для работы с Torexxx Creator?

Абсолютно нет! Torexxx Creator разработан специально для людей без технического образования. Весь процесс создания сайта происходит в визуальном редакторе с помощью перетаскивания элементов.

Можно ли экспортировать код созданного сайта?

Да, начиная с тарифа "Профессиональный", вы можете экспортировать чистый HTML, CSS и JavaScript код вашего проекта для самостоятельной публикации или дальнейшей доработки.

Предоставляете ли вы хостинг для созданных сайтов?

Да, все тарифные планы включают бесплатный хостинг на наших серверах. Вы также можете подключить свой собственный домен начиная с тарифа "Профессиональный".

Можно ли создавать интернет-магазины с помощью Torexxx Creator?

Да, конечно! В Torexxx Creator есть специальные шаблоны и модули для создания интернет-магазинов с корзиной, каталогом товаров и интеграцией с платежными системами.

Как работает бесплатный тариф?

Бесплатный тариф позволяет создать до 3 проектов с использованием базовых шаблонов. Ваши сайты будут доступны по поддомену вида ваш-сайт.torexxx-creator.com. Это отличный способ опробовать платформу перед переходом на платный тариф.
